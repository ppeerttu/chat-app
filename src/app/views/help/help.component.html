1. Registration and logging in

a. Registration
Registration into the service is done by filling a registration form. The form can be foud when clicking the "Register" button at the Login -page. The form has to be filled with valid data. When the form is filled up correctly, you can click "Register". You will be notified whether or not the registration is successful. You can get to the Login page by pressing the "Back to login" button at the login page. More specific information about required user information below. Please note that no user information is used in anything other than access control; not a single email will be sent from this software. You can fill in falsy user information if you want to.

Data Required Min length Max length Additional info
Username Yes 3 15 Unique
First name Yes 1 15 Private
Last name Yes 1 15 Private
Email Yes - 15 Valid email
Password Yes 6 20 Will be confirmed with another form field

b. Logging in
Logging into the service can be done at Login -page. This is the default page of the site. Log in to the service by filling in the username and password of your account. Click then the "Login" button. If the logging in is successful, you will be redirected to the Chat -page. If login is unsuccessful, you will be notified about it.

2. Managing the rooms


NOTE: In following sub-sections, you will be assumed to be in the Chat view.

a. Joining a room
You can join a room from the left sidebar. Open the sidebar by clicking the Menu button in the top left corner. Select "Join rooms", which will open a list of rooms. You can join any room by clicking the plus (+) button. If there is only minus (-) button next to some room's name, you are already in that room. If the room you're joining to is private room, it is protected by a password. You will be shown a dialog where you can insert the password. Join the room by clicking "Join" or cancel the action by clicking "Cancel". When you have joined into the room, you can send and receive messages in it. You can join as many rooms as you like.

b. Leaving a room
You can leave a room by opening the sidebar by clicking the Menu button from top left corner of the screen. Select "Join rooms" from the sidebar and a list of rooms will open in the sidebar. Leave any room you're in by clicking the minus (-) button next to the room name. You can join the room again immediately if you want to.

c. Creating a room
You can create a room by opening the sidebar by clicking the Menu button from top left corner of the screen. Select "Create a room" from the sidebar and a form will be opened. Fill the name of the room into the form and fill password for the room if you want the room to be private. After you're set, click "Create" button. A notification will be shown in the bottom of the screen whether or not the room was created successfully.

3. Sending a message


NOTE: In following sub-sections, you will be assumed to be in the Chat view.

a. Selecting a room
You can select a room by clicking the room in "My rooms" list. The list is located in the left side of the screen on larger screens, and in the right sidebar on small screens (mobile).

b. Sending a message in a room
You can send a message when you're in a room (section 3.a). Send a message by typing the message in the message form field at the bottom of the screen and pressing Enter or send button next to the message form field.

4. Managing the user profile


NOTE: In following sub-sections, you will be assumed to be in the Chat view.

a. Changing the user information
To change the user information, open the left sidebar by clicking the Menu button from the top left corner of the screen. Select "Settings" from the left sidebar and "Profile" under the settings. A profile modal will be opened for you. You can change any user information as long as you follow the validation rules explained at section 1.a. You can cancel the action by clicking "Cancel".

b. Changing the password
Open the profile modal (see section 4.a). If you want to change only password, do not touch any other form field than "Password" and "Confirm password". Insert the new password into both fields. If you want to, you can also change other user information. Save the information by clicking "Save" or cancel the action by clicking "Cancel".

5.About the software and help

a. About the software
You can get information about the software by going to the About -page. You can get to the About -page from any view in the software:

In the ogin- and Register -views, click the "About" -link at the bottom right of the page.

In the Chat -view you'll get to the About -page by opening the left sidebar, clicking the "Settings" and selecting "About" from the settings list.

b. Help
Open the "Settings" list (see section 5.a above). Select "Help" from the list which will open the Help page in a new tab.

6. Common error messages


Common error messages and usual explanations for them are listed below.

  • Login unsuccessful
  • Logging in was unsuccessful. This happens usually due to wrong user credentials, lost of internet connection or unexpected server error.
  • Registration failed
  • Registration was unsuccessful. This happens usually when the username is already taken, the internet connection is lost or unexpected server error.
  • Room creation failed
  • Room creation failed. Either the room name was already taken, the internet connection is lost, or an unexpected server error has occurred.
  • Joining the room failed
  • Joining the selected room failed. This can be caused by wrong password (private rooms), lost internet connection or unexpected server error.
  • Updating user data failed
  • User data was updated unsuccessfully. This can be caused by already taken username (when changing the username), lost internet connection or unexpected server error.

7. Terms


Common terms related to this service and their explanations are listed below.

  • User is a private person, who can register to this service and use it
  • Room is a virtual space, where users can send messages to each other (see section 2).
  • Chosen room is a room that a user has selected in the Chat -view for current chat room (see section 2.a)
  • My rooms are the rooms the user is joined in currently
  • Left sidebar is a menu that can be opened in the Chat -view from the Menu button in the top left corner of the page
  • Participants is a list of users that are in the room currently selected as chosen room
  • Room info contains information about chosen room
  • Modal is a window that is opened on top of the page
  • Dialog is a window that is opened on top of the page and will provide some information or ask for confirmation from the user
  • Message is a string sent in a room and other participants in that particular room can see it
