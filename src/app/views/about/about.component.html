Info

This is a chat application software. This software is made as a hobby, and it is by no means commercial.

Looking for help?

User data

This software uses user information only to indentify users. Thus, no data is collected after registration about users. You do not need to submit your real name or email address. This software will not use the email addresses submitted by users in any way. No emails will be sent by this sofware. If you want your user data to be deleted from this software, please contact to the author.

Conversations

Conversations via this application are not guaranteed to be secure. Thus, do not send any confidential information in any rooms in this application. Messages are not collected in to any database at the moment. However, this may change in the future.
